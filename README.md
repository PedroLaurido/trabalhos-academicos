# Trabalhos Académicos

Repositório com **trabalhos e projetos realizados durante o meu percurso académico**.  
O objetivo é centralizar e documentar a experiência prática adquirida durante o meu percurso no Ensino Superior.

---

## Objetivos
- **Portfólio académico:** mostrar projetos académicos que foram desenvolvidos.
- **Organização:** reunir os trabalhos num só local.
- **Experiência:** apresentar contacto/experiência com determinadas linguagens, ferramentas e tecnologias de programação
- **Melhoria futura:** atualizar e refatorar com o tempo.

---

## Estrutura do Repositório
**Exemplo:**
```
/trabalhos-academicos/
├── unidade-curricular1/
│   ├── trabalho1/
│   └── .../
├── unidade-curricular2/
│   ├── .../
├── unidade-curricular3/
│   ├── .../
├── .../
└── README.md
```

Cada pasta contém um `README.md` com descrição, objetivos, tecnologias e instruções de execução.
